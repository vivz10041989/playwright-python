This is a dummy project created to build an end to end automated framework using below tools. 
This framework will trigger Github actions to run test scripts when we push any changes to Main branch.
The generated test results are posted to the Test rail tool as a test run.
This framwework will send email notification about the job status.
Below tools and technologies are used to acheive this -
1. Test management tool - TestRail
2. IDE - cursor
3. Programming language - Python
4. BDD Style script development
5. Version control - Github
6. CI/CD - Github actions
